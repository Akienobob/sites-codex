# Старый Икарус — концепт, роли и эскизы

## 1) Роли команды
- **Developer** — HTML/CSS/JS реализация, структура страниц, анимации и адаптив.
- **Designer** — художественный стиль, логотип, типографика, композиция блоков.
- **QA** — проверка контента, ссылок, доступности и визуальной целостности.
- **Integrator** — сборка целостного статического сайта и контроль соответствия ТЗ.

## 2) Дизайн-концепт
- База: чёрный/графит/белый/серый.
- Тёплые акценты: только солнце и световые следы (жёлтый/оранжевый/красный).
- Образ: детализированный крылатый автобус Икарус с крыльями по бокам кузова (как у птицы), направленный к солнцу.
- Атмосфера: философская, контрастная, с древнегреческими мотивами через ритм и рамки.

## 3) Эскизы
- `docs/sketches/draft-a-svet-ellady.svg` — классический герой-блок с реалистичным боковым силуэтом автобуса.
- `docs/sketches/draft-b-polet-ikarusa.svg` — динамичный вариант с траекторией полёта.

## 4) Реализованный статический сайт
- Страницы: `index.html`, `about.html`, `music.html`, `media.html`, `contacts.html`.
- `Impressum` (только DE, вне главного меню): `impressum.html`.
- RU/EN переключение: кнопка в шапке (на главной и остальных страницах), управление через `script.js` и `localStorage`.
- Общие стили и композиция: `styles.css`.
- Логотип: `assets/logo-winged-ikarus.svg`.

## 5) Как открыть локально
```bash
python -m http.server 8000 --bind 0.0.0.0
# затем открыть http://127.0.0.1:8000/
```
