# Проект: статический сайт группы «Старый Икарус»

## 1) Роли команды

### Developer
**Зона ответственности:**
- Реализация статического сайта (HTML/CSS/JS) с адаптивной вёрсткой.
- Подготовка структуры страниц: `Главная`, `О группе`, `Музыка`, `Медиа`, `Контакты`, а также скрытая от меню страница `Impressum` (только DE).
- Реализация переключения языков (RU/EN) на главной странице.
- Интеграция анимации логотипа (автобус с крыльями, движение слева направо к солнцу).

**Артефакты:**
- Файловая структура проекта.
- Семантическая HTML-разметка.
- Стили, анимации, базовая доступность (a11y).

### Designer
**Зона ответственности:**
- Визуальная концепция в светлых тонах с древнегреческими мотивами.
- Подбор типографики, цветовой палитры и графических паттернов.
- Проектирование UI-компонентов: hero-блок, карточки треков, медиагалерея, контакты, футер.

**Артефакты:**
- Moodboard и UI-направления.
- Наброски макетов (wireframes) для ключевых страниц.
- Спецификация визуального стиля.

### QA
**Зона ответственности:**
- Проверка корректности отображения всех страниц и блоков.
- Проверка переключения языков RU/EN.
- Проверка наличия ссылки `Impressum` в футере и отсутствия её в главном меню.
- Кросс-браузерная и адаптивная проверка.

**Артефакты:**
- Чек-листы тестирования.
- Отчёты о дефектах и статусах исправлений.

### Integrator
**Зона ответственности:**
- Оркестрация задач между ролями.
- Контроль консистентности контента RU/EN и соответствия бренду.
- Подготовка итоговой сборки и релиз-готового варианта.

**Артефакты:**
- Дорожная карта этапов.
- Релизный checklist.

## 2) Дизайн-концепт (монохромный, философский, с античными мотивами)

### Базовая идея
Сайт передаёт образ «дорожной философии»: крылатый автобус как метафора пути, солнце как поиск смысла, античные орнаменты как символ вечных вопросов. Стиль — контрастный монохром (чёрный/белый/серый) с тёплым солнечным акцентом.

### Цветовая палитра (основа)
- `#0F1012` — основной чёрный фон.
- `#1F2125` — графитовый фон блоков.
- `#F1F1F1` — светлый контрастный текст/карточки.
- `#8A8E96` — металлический серый для деталей логотипа.
- `#FFE47A` / `#FF944A` / `#FF4F30` — тёплый солнечный спектр (единственный яркий акцент).

### Типографика
- Заголовки: элегантный шрифт с классическим характером (serif).
- Текст: нейтральный читаемый sans-serif.
- Декоративные элементы (цитаты): курсивный serif.

### Ключевые визуальные мотивы
- Греческий меандр (тонкая рамка/разделители).
- Полупрозрачные солнечные круги в фоне.
- Текстурный «бумажный» фон.
- Анимированный логотип: автобус с крыльями, движение к солнцу слева направо.

## 3) Набросок A — «Свет Эллады» (более классический)

### Главная
1. **Hero:**
   - Крупный логотип (автобус + крылья + солнце).
   - Слоган RU/EN.
   - Кнопка переключения языка (RU | EN) в верхней части hero.
2. **Блок «Философия группы»:** короткий манифест + цитата.
3. **Блок «Музыка»:** 3–4 релиза карточками.
4. **Блок «Медиа»:** сетка фото/видео-превью.
5. **Блок «Контакты»:** email, соцсети, форма/кнопка связи.
6. **Футер:** копирайт + ссылка на `Impressum`.

### Визуальная характеристика
- Много «воздуха», мягкие тени.
- Чёткая центральная композиция.
- Декор — дозированно, через разделители и фоновые орнаменты.

---

## 4) Набросок B — «Полет Икаруса» (более современный и динамичный)

### Главная
1. **Hero-панорама:**
   - Логотип в движении (автобус проходит по дуге).
   - Солнце как динамический градиентный круг.
   - Два CTA: «Слушать» / «О группе».
   - Переключатель языков RU/EN в правом верхнем углу.
2. **Лента жанров:** folk-rock / blues / jazz / psychedelic (анимированные плашки).
3. **«Глубокие тексты»:** интерактивные цитаты на карточках.
4. **Медиа-карусель:** фото сцены/репетиций + live snippets.
5. **Контакты и футер:** минималистично, `Impressum` только внизу.

### Визуальная характеристика
- Больше движения и микровзаимодействий.
- Контрастные акценты на светлом фоне.
- Орнаменты — как геометрические маски/рамки.

## 5) Предложение к следующему этапу (до начала разработки)

Перед кодингом предлагается утвердить:
1. Один из набросков (A или B) как основной.
2. Финальную палитру и набор шрифтов.
3. Сценарий анимации логотипа (скорость, траектория, интенсивность).
4. Структуру контента RU/EN для каждой страницы.

После утверждения — переходим к реализации статического сайта.

## 7) Где находятся файлы и как их открыть

### Расположение
- `docs/sketches/draft-a-svet-ellady.svg`
- `docs/sketches/draft-b-polet-ikarusa.svg`

### Быстрый просмотр из терминала
```bash
# проверить, что файлы есть
ls -la docs/sketches
```

### Открыть в браузере (Linux)
```bash
xdg-open docs/sketches/draft-a-svet-ellady.svg
xdg-open docs/sketches/draft-b-polet-ikarusa.svg
```

### Если работаете в VS Code
1. Откройте папку проекта `sites-codex`.
2. В проводнике перейдите в `docs/sketches/`.
3. Кликните по SVG-файлу — откроется превью.

### Открыть как локальный файл в браузере вручную
Можно перетащить SVG-файл в окно браузера или открыть по пути `file:///.../docs/sketches/*.svg`.


## 8) Обновление по качеству эскизов

- Логотип в обоих набросках переработан в более детальный художественный знак: узнаваемый корпус автобуса, окна, фары/сигналы, объёмные крылья с перьями, траектория полёта к солнцу.
- Цветовые схемы A и B полностью убраны из жёлто-коричневой и синей базы; оставлен монохромный фундамент (чёрный, графит, белый, серый).
- Тёплые цвета применяются только в зоне солнца и световых следов (жёлтый/оранжевый/красный).
